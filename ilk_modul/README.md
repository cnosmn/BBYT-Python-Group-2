# Python Algoritmik Problem Çözme Dersi

Bu repository, Python programlama dilini algoritmik problem çözme odaklı olarak öğreten kapsamlı bir eğitim materyalidir.

## 📚 Müfredat İçeriği

12 haftalık müfredat şunları kapsar:
- **Temel Python:** Değişkenler, veri tipleri, operatörler
- **Karar Yapıları:** If-else, koşullu algoritmalar
- **Döngüler:** While, for, iç içe döngüler
- **Veri Yapıları:** Listeler, string işlemleri, sözlükler
- **Fonksiyonlar:** Modüler programlama, recursion
- **Dosya İşlemleri:** Veri okuma/yazma
- **İleri Algoritmalar:** Sıralama, arama, hash tabanlı yapılar
- **Final Projesi:** Algoritma odaklı uygulamalar

## 📁 Dosya Yapısı

```
├── ders1.py - ders14.py    # Haftalık ders kodları
├── odev1.py - odev13.py    # Haftalık ödev çözümleri
├── mufredat.md            # Detaylı müfredat
└── README.md              # Bu dosya
```

## 🚀 Nasıl Kullanılır

1. **Python Kurulumu:** Python 3.x sürümü gereklidir
2. **Dersleri Takip:** `ders1.py` ile başlayarak sırayla çalışın
3. **Ödevleri Çözün:** Her ders sonrası ilgili ödevi tamamlayın
4. **Kodları Çalıştırın:** Terminal/Command Prompt'ta `python ders1.py` gibi çalıştırın
